# Projeto FrameWorks de Front-End

Projeto **MC Calçados** tem como escopo ser uma aplicação tanto para cliente visualizarem, quanto para funcionario  organizarem produtos.

## Tecnologias

- Vue.js
- Tailwind
- JsonServer

## Como rodar o projeto

```sh
npm install
```

### Rodar Front-end

```sh
npm run dev
```

### Rodar Back-end

```sh
npm run server
```
